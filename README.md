# Grade Average Calculator (Python)

## Overview
This application is a Python based calculator designed to compute the average of multiple user entered grades. The program collects input dynamically, validates entries, and calculates the final average with structured logic.

## Features
- Accepts multiple grade inputs
- Performs real-time average calculation
- Input validation to prevent invalid entries
- Handles edge cases such as empty input or non numeric values

## Technologies Used
- Python 

## Skills Demonstrated
- Loop structures
- Conditional logic
- Input validation
- Error handling
- Basic algorithm design

## How to Run
1. Ensure Python is installed.
2. Run the program:
   python main.py
